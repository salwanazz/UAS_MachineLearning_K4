Sistem Deteksi Spam Email Berbasis Website merupakan aplikasi web yang dikembangkan dalam mata kuliah Machine Learning untuk mengklasifikasikan email ke dalam kategori Spam dan Non-Spam (Ham) secara otomatis. Sistem ini menerapkan pendekatan supervised learning sebagai bagian dari machine learning serta didukung oleh teknik Natural Language Processing dalam pengolahan teks.
Aplikasi ini membantu pengguna menyaring email berbahaya seperti spam, penipuan, dan phishing dengan memanfaatkan model machine learning yang telah dilatih dan siap digunakan pada tahap implementasi sistem.
